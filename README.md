## 项目名称

活动助手Activity Helper

## 项目介绍

在高校，各个学生社团和学生组织都会在各个时间发布各式各样的活动信息，但是目前缺乏一个可靠、同一、简便的平台汇总各类的活动信息。《BJUT活动助手》是基于北京工业大学校园内的各项信息开发的一款集报名、统计、汇总、收藏于一体的活动助手小程序，旨在帮助高校统筹管理各类校园活动，以提升学生和老师的活动参与度。
